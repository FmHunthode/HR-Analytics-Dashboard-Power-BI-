# HR-Analytics-Dashboard-Power-BI-
This project analyzes workforce data to provide insights into employee demographics, performance, promotion eligibility, retrenchment risk, and overall HR metrics. The dashboard helps HR managers make data-driven decisions regarding workforce planning and talent management.
## Dataset Overview
The dataset contains employee information including:

Employee ID

Department

Gender

Age

Years of Service

Performance Rating

Promotion Status

Retrenchment Status

Salary Information
